# 🛍️ Customer Segmentation using K-Means Clustering

## 📌 Project Overview

This project segments mall customers into meaningful groups using **K-Means Clustering** based on their purchasing behavior. The objective is to help businesses understand different customer segments and create personalized marketing strategies.

---

## 🎯 Objectives

- Perform customer segmentation using K-Means Clustering.
- Identify the optimal number of clusters using the Elbow Method.
- Visualize customer segments.
- Apply PCA for dimensionality reduction and visualization.
- Generate meaningful business insights from the clustered data.

---

## 📂 Dataset

- **Dataset:** Mall Customers Dataset
- **Records:** 200 Customers

### Features Used

- Age
- Annual Income (k$)
- Spending Score (1–100)

---

## 📈 Results

- Optimal Number of Clusters: **5**
- Successfully segmented customers into five distinct groups.
- Visualized customer clusters using K-Means and PCA.
- Generated business-oriented insights for targeted marketing strategies.

---

## 💼 Business Insights
